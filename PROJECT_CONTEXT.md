# Portfolio Project Context

## Project Overview
A modern, minimal, and accessible portfolio site for Sai Sankar Madeti, focused on AI-powered services and solutions. The site is designed with inspiration from Apple and other leading tech brands, emphasizing whitespace, product-centric visuals, strong typography, and smooth motion.

## Key Features & Design Principles
- **Minimalism & Negative Space:** Generous spacing, uncluttered layout, and focus on content.
- **Product-Centric Visuals:** Subtle hero background, soft gradients, and clean service cards.
- **Typography:** Uses Inter and SF Pro Display for clear hierarchy and modern feel.
- **Navigation:** Sticky navbar with smooth hover/active effects and accessibility labels.
- **CTAs:** Bold, high-contrast buttons with ample breathing room and focus states.
- **Motion:** Smooth, scroll-triggered animations for cards and sections.
- **Accessibility:** Strong color contrast, aria-labels, keyboard focus, and semantic HTML.
- **Performance:** Modern image formats, responsive design, and optimized mobile spacing.
- **No em dashes or hyphens as separators:** All visible text uses natural language and punctuation.

## Main Sections
- **Hero:** Name, tagline, and stats.
- **Services:** 10 AI-focused offerings, each with icon, description, and tech stack.
- **Why AI:** Market opportunity and value stats.
- **Process:** 4-step engagement process.
- **Contact:** Email, LinkedIn, and GitHub links.
- **Footer:** Copyright.

## Tech Stack
- Pure HTML, CSS, and JS (no frameworks)
- Served locally via Python's http.server for development

## Accessibility & Performance
- All navigation and buttons have aria-labels
- Color contrast meets WCAG standards
- Responsive and mobile-friendly
- Hero background uses WebP for performance
